I performed both the (1) direct-Zc-calculation method ($Z_c$ is directly calculated during the fit) and the (2) self-consistent-loop method ($Z_c$ is fixed during the fit and only gets updated after the fit). Note that all the following analysis is done in my python code, no matlab code is involved. Therefore, apart from verifying the two methods give identical results, I also need to verify that the python code gives the same results as the old matlab code.


In the fitting process:

  • $U$ is the only free parameter for the fit.

  • $t^{\prime\prime}$ is fixed throughout the entire script. denoted as tppt in the code.

  • $\frac{t}{U}$ and $\frac{t^\prime}{t}$ are being looped over. denoted as tU and tpt in the code. In each step of the loop, these two parameters are also fixed. within this step, we fit the data with the only free parameter $U$. With the resulting $U$, we calculate the $\chi^2$. In effect, $\chi^2(\frac{t}{U}, \frac{t^\prime}{t})$ is a function of $\frac{t}{U}$ and $\frac{t^\prime}{t}$.

  • At the end of the fit, we plot out the $\chi^2(\frac{t}{U}, \frac{t^\prime}{t})$ landscape, and pick the best parameters set.
